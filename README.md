# Network Monitoring & Incident Dashboard

A web-based application designed to simulate and manage network infrastructure monitoring and incident tracking.

## 🚀 Overview

This project demonstrates practical knowledge of network environments, infrastructure monitoring, and incident management, combining concepts from telecommunications and IT infrastructure.

The system provides visibility into network device status, allowing users to simulate failures and track incidents in a centralized dashboard.

## 🧠 Purpose

This project was developed to represent real-world scenarios in network operations, focusing on:

- Infrastructure monitoring
- Incident response
- System reliability
- Network troubleshooting workflows

It aligns with professional environments in telecom, IT infrastructure, and network engineering.

## ⚙️ Features

- 📡 Device Monitoring (Online/Offline status)
- ⚠️ Incident Simulation and Tracking
- 📊 Dashboard Visualization
- 🧾 Incident Log History
- 🔄 Real-time Status Updates

## 🏗️ Architecture

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js (Express)
- **Data Storage:** JSON (simulated database)
